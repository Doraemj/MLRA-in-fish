# MLRA-in-fish
This is the code repository to predict the BCFs in pollutants transporting during the processing process.
It works for:
Supplementary data and code for Assessing human health risks of organic contaminants in fish: A modeling approach integrating chemical bioconcentration and food processing
https://doi.org/10.1016/j.toxrep.2024.101805

# Introduction
In this project, we built a fuzzy rsik assessing model for predicting chemical exposure from contaminated fish. We use machine learning method to predict the bioconcentration factors, and SHARP method was used to explain the effect of the indicators.
